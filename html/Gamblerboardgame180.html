Gambler is a board game originally made by Parker Brothers in 1975.

A redesigned version of the game was released in 1977. The game is now out of print, and the US trademark on the game was cancelled in 1983.[1]

Game[edit]

Source: Gambler official rules

The game consists of players taking turns rolling dice to move around the game board. Events giving players a chance to win money can happen when a player lands on a space, draws cards, or rolls doubles. The object of the game is to be the first player to win $1,000 of in-game money.

Some contests in the game use a special "Dice Shaker", a device containing six dice with an area to shuffle the dice and a narrow handle that holds each of the dice in a line. In these contests, all players make some type of bet and then a player shakes the Dice Shaker and moves the dice into the handle. The order of the dice in the handle determines which (if any) player wins the contest.
